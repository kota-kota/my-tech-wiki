# VSCodeショートカットキー

## 基本

| キー             | 説明                     |
| ---------------- | ------------------------ |
| Ctrl + p         | ファイルを検索する。     |
| Ctrl + Shift + p | コマンドパレットを開く。 |

## エディタ

| キー                     | 説明                            |
| ------------------------ | ------------------------------- |
| Shift + Alt + スクロール | 矩形選択。                      |
| Ctrl + PageDown/PageUp   | 次のファイル/前のファイルへ移動 |
| Ctrl + Click (F12)       | 定義へ移動する。                |
| Alt + ←/→                | 戻る/進む。                     |
